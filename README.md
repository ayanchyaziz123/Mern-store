## Frontend
1. [react](https://reactjs.org/)

## Database
1. [mongoDB](mongodb) online

## Prerequisites
1. Java Script
2. Nosql


install node js in your machine

## Installation backend
go to backend folder and open cmd here
```
npm i
```
## Installation frontend
go to frontend folder and open cmd here
```
npm i
```
